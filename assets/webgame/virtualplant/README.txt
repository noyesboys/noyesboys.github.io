Cannabis plant asset overhaul

Contents:
- cleaned/: original files from the uploaded archive, cleaned, centered, and with stray label artifacts removed.
- base_stages/: base stage set with seedling1-10, vegative1-10, flowering1-10.
- combined_variants/: generated per-stage variants with names like seedling3yellow3.png, vegative3stretch3.png, flowering4indica4.png.
- transitions/: smoother blend frames between seedling10 -> vegative1 and vegative10 -> flowering1.

Naming:
- yellow = yellowing
- stretch = stretched / elongated
- indica = shorter, broader indica-style form
- sativa = taller, slimmer sativa-style form
- top1 = topped once
- top2 = topped twice
